# Ever Popup v1.2.2

Module pour Prestashop 1.6.

Ce module permet l'affichage d'un popup dont le contenu est paramétrable.

- Correction typo dans formulaire de configuration,

- Traduction du module en-fr,
- Ajout du README.md.
